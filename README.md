# Breakout Clone
## Description
This is a clone of the 80s hit game Breakout i made for the assignment on Day 87 of the 100 Days of Code course. 


## Installation
- Clone repository, create virtual environment and install dependencies:
```bash
git clone https://github.com/wnsnk/breakout-clone
cd breakout-clone
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

- run the app
```bash
python main.py
```
